# 自動販売機の売り上げ管理システム

larabaelで作成した管理システム

# Requirement

- docker
- docker-compose
- php
- larabel
- mylsql
- phpadmin


# Usage

docker-composeで立ち上げ
コンテナ内に入り操作

```bash
git clone https://github.com/kazun1224/laravel_management_system.git
cd examples
docker-compose up -d
docker-compose exec app bash

root/:/var/www/# ...
```
